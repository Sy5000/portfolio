Database + UI Design

Brief

Summer escapes is a small startup who's goal is to inform visitors about holiday destinations using a library of user generated content, their long term goal is to promote travel bookings via the site.

Users can submit reviews, ratings and holiday photos. Aggregate data is also displayed to give visitors a broader idea of the trends across the platform. eg Top 10 destinations by: most reviewed / most clicked / average score. Staff can also administrate user submitted content via a simple secure custom CMS.

Challenge

Develop a physical database model and UI to suit the needs of the summer escapes website.

Solution

A robust and reliable storage solution was deployed in the form of a 'back-end' & relational database system. The schema was custom designed and built to meet the requirements of the project by analysing the relationship between the business rules, the platform functionality and the security considerations.

Skills

I have learned how to organise and store complicated datasets efficiently using a relational database system. On a practical level I exercised a combination of programming concepts and database querying to perform complicated actions on data using server side scripting. I can confidently develop & implement conceptual, logical and practical database models and security measures to ensure data integrity.
