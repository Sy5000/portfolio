Dynamic Site with Custom C.M.S

Brief

'Fresh Prints ' is a mid size screen printers who specialise in selling hand made designs and merchandise which are made to order. Their website allows customers can browse the available merchandise options (styles, size, colour) and place an order from the stock available. Alongside the order customers have the option to upload a design by attaching a design file.

Stock / stock levels & customer orders are managed via secure access to a back-end CMS / admin area which was built to meet the specific needs of the business.

Research

The UI and UX journey was developed using a meticulous market research process which was 'e-commerce' focused. This included market segmentation, gathering expectations / ideas through group interviews, distilling feedback into valuable user 'personas' & workshopping solutions to common user problems.

User stories were then developed to guide the behaviour and structure of the website. An Interactive prototype was then built to test and refine the UI/UX design against user demographics before the production stage.

Build and Testing

A full stack custom CMS system was built from the ground up which could facilitate each of the many business needs (secure login, secure admin area, stock level & inventory management, order processing, data validation & sanitisation, 1 user account per email, valid design file format + size limitations).

Use cases and data requirements were gathered and the database structure was determined through a process of normalisation.

Tactile user feedback is provided on the order form by checking the quantity required to fulfil the order against the current available stock levels (AJAX request). The user is alerted if the order is too large and the order form is temporarily disabled until order quantity < stock levels.

Each block of functionality was thoroughly tested and documented and each layout was tested against web accessibility guidelines before deployment.
